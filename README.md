# 🔐 Password Strength Checker

## Project Overview
This is a simple Python project that checks how strong a password is. It was built as part of DecodeLabs Project 1 (Batch 2026).

The goal of this project is to help users understand whether their password is weak, moderate, or strong based on basic security rules.

---

## What it does
When you enter a password, the program checks:

- Length of the password  
- Whether it contains uppercase letters  
- Whether it contains lowercase letters  
- Whether it contains numbers  
- Whether it contains special characters  

Based on these checks, it gives a strength rating:

- 🔴 Weak  
- 🟡 Moderate  
- 🟢 Strong  

It also gives suggestions to improve weak passwords.

---

## Features
- Simple user input system  
- Password confirmation check  
- Strength evaluation logic  
- Secure comparison using constant-time check  
- Helpful feedback for improvement
